Урок №21 (Основи JavaScript. Частина №1)

Задача №1

Яка назва змінної вірна?

  1. let user-name
  2. let UserName
  3. let userName

Відповідь: варіант №3 - let userName

У JS для запису змінних використовують стиль lowerCamelCase.

Задача №2

Чи буде помилка при виконанні наступної програми?

const userName = 20
userName = 'Іштван'

Відповідь: Так.

Неможливо призначати константі значення після її декларації.

TypeError: invalid assignment to const 'userName'

Задача №3

Що потрапить в консоль?

let numOne = 20
let numTwo = '20'
let numSum = numOne + numTwo
console.log(numSum)

Відповідь: '2020'.

Один з операндів є рядком, тож інший операнд було конвертовано в цей самий тип.

P.S. У браузерній консолі також є завдання та рішення.
